Update an Existing Assessment in Library
========================================

To update assessments in the library, follow these steps:

1. Choose one of the following methods to find the assessment you want to update:

.. tab-set::

   .. tab-item:: From the Guide Editor

      Click the **Edit: <Assessment Name>** button to the right of the assessment.

      .. image:: /img/guides/editassessmentbutton.png
         :alt: Edit Assessment
         :width: 450px

   .. tab-item:: From the Assessment List

      Click the **Assessment** button to view all assessments, then click the assessment to open it.

      .. image:: /img/guides/editassessmentlist.png
         :alt: Edit Assessment List



3. To update the assessment, open the assessment, and click **Update in Library** in the lower right corner.

   .. image:: /img/guides/UpdateInLib.png
      :alt: Update in Library
      :width: 450px
