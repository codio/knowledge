Cursor Features
===============

Multiple Cursors
----------------
You can place multiple cursors anywhere in a file using the following keyboard shortcut:

- **Alt+Click** (Windows)
- **Option+Click** (Mac)

Click to place the cursor at the desired location, and then press and hold **Alt** or **Option** while clicking at new locations. When you release Alt/Option, you can start typing and the content will be duplicated at all cursor locations.
